# 6. Lab：AI Model Prompt Comparison

> Module 1 动手实验——通过对比实验，建立"什么时候用什么"的直觉。

---

## 实验目的

亲手对比三种模式的输出差异，形成使用直觉：

| 对比维度 | 你会看到什么 |
|----------|-------------|
| 有 Web Search vs 无 Web Search | 同一个问题，输出质量天差地别 |
| Web Search vs Deep Research | 简单回答 vs 深度综合报告 |
| 有上传文件 vs 无上传文件 | 通用回答 vs 基于你资料的精准回答 |
| 有 typo vs 无 typo | 结果几乎一样（AI 能容忍错别字） |

---

## 实验清单

### 实验 1：Web Search 开 vs 关

| Prompt | 无搜索 | 有搜索 |
|--------|--------|--------|
| "什么是 6 7 梗？" | ❌ 不知道 | ✅ 解释梗的来源和用法 |
| "附近评分高的健身房" | ❌ 泛泛而谈 | ✅ 给出具体地点 |
| "下一部复联什么时候上映？" | ❌ 可能过时 | ✅ 最新档期 |
| "今天美国有什么大新闻？" | ❌ 无法回答 | ✅ 实时新闻 |

**结论：** 涉及时效性、地域性、冷门信息 → 开搜索。

---

### 实验 2：Web Search vs Deep Research

| Prompt | Web Search | Deep Research |
|--------|-----------|---------------|
| "灰色市场肽类安全吗？请用高质量来源" | 几个来源的简短回答 | 多来源综合报告 + 引用 |

**结论：** 简单查询用搜索，需要综合分析用 Deep Research。

---

### 实验 3：上传文件的效果

```
Prompt："我能把火箭推进的怪兽卡车放在车库里吗？"

无文件 → AI 给通用法律建议
上传租约 → AI 根据你的具体租约条款回答（可能发现禁止条款）
```

**结论：** 涉及你的私人情况，上传相关文件 = 精准回答。

---

### 实验 4：Typo 容忍度

```
有 typo："wy do cats star at wals lke theyr seein ghsts?"
无 typo："Why do cats stare at walls like they're seeing ghosts?"

结果 → 几乎一样
```

**结论：** 打字快有错别字没关系，不用花时间改。

## 自己试试

```
建议尝试的自定义 Prompt：
- "今天 [你的城市] 适合野餐吗？"  → 对比有无搜索
- 上传一份你的文档 → 问 AI 基于文档回答问题
- 用你最关心的话题 → 对比 Web Search vs Deep Research
```

## Module 1 完整回顾

```
1. AI Novice vs Power User   → 差距在提问方式
2. Pretrained Knowledge       → AI 知识的来源和边界
3. Web Search                 → 获取截止日期后的新信息
4. Web Search Sources         → 引导 AI 用靠谱来源
5. Deep Research              → 复杂问题的深度综合分析
6. Lab（本节）                → 动手对比，建立直觉
```

**Module 1 核心收获：**

> 信息获取有三条路径（预训练 → 搜索 → 深度研究），
> 选哪条取决于**问题复杂度 × 时效性要求 × 你愿意等多久**。
