# 2. Pretrained Knowledge

> AI 的知识来自互联网——理解它"读过什么、读了多少"，你就能预判它什么时候靠谱、什么时候不靠谱。

---

## AI 的知识从哪来？

AI 通过阅读互联网上的海量文本来"学习"：

| 来源 | 举例 |
|------|------|
| 社交论坛 | Reddit、Quora 上的问答 |
| 书籍 | 各领域出版物 |
| 百科 | Wikipedia |
| 新闻 | 各大新闻网站 |
| 学术论文 | 研究文献 |
| 代码 | GitHub 等代码库 |

> 这些来源加在一起有**数万亿词**，统称为 **预训练知识（Pretrained Knowledge）**。

---

## 关键规律：频率决定可靠性

**互联网上出现得越多的内容，AI 回答越准确。**

| 频率 | 话题举例 | AI 表现 |
|------|----------|---------|
| 🟢 极高 | 烹饪、电影、名人、常识 | 非常可靠 |
| 🟡 中等 | 专业领域（天文学、材料学） | 基本准确，偶有遗漏 |
| 🔴 极低 | 粤语内容（<0.1% 互联网数据） | 容易出错 |
| ⚫ 零 | 你公司的内部机密数据 | 完全不知道 |

**实操判断法：**

```
问自己：这个问题在互联网上能搜到多少相关内容？
  → 很多 → 放心问
  → 很少 → 结果要核实
  → 没有 → 必须你提供资料
```

---

## AI 因此具备的能力

| 能力 | 举例 |
|------|------|
| 回答广泛话题 | "手机掉汤里怎么办？" |
| 理解冷门知识 | "旅行者 1 号唱片上录了什么？" |
| 容忍打字错误 | "can u cook egs in microwve?" → 完全能理解 |

> 💡 **不用花时间纠正每个错别字。** AI 见过大量拼错的文本，打字快有 typo 也完全没问题。

---

## AI 的知识局限

| 局限 | 原因 | 对策 |
|------|------|------|
| **有截止日期** | 训练在某天停止 | 需要时让它做 Web Search |
| **反映误解** | 互联网有错误信息 | 重要结论要验证 |
| **有过时信息** | 训练数据可能老旧 | 要求"基于最新信息" |
| **不知道你的私有数据** | 从没见过 | 必须你上传/粘贴 |

---

## 本节要记住的 3 点

1. **AI 的知识 = 它读过的互联网文本** — 常见话题准、冷门话题可能猜
2. **频率 = 可靠度** — 问之前想想"这内容在网上多不多"
3. **预训练知识有边界** — 新信息、私有数据、冷门语言需要额外补充
